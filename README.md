## LAB1 - HelloToast App 

# Description

Ce projet est une application Android simple développée avec Android Studio en Java.
Elle permet d’afficher un message Toast et d’incrémenter un compteur.

# Objectifs
- Comprendre la structure d’un projet Android
- Manipuler les composants UI (TextView, Button)
- Gérer les événements (click)
- Mettre à jour dynamiquement l’interface

# Technologies utilisées
  - Java
  - Android Studio
  - XML (interface utilisateur)

# Fonctionnalités
  - Bouton "Afficher un message" → affiche un Toast
  - Bouton "Incrémenter le compteur" → augmente la valeur affichée

# Interface
  - Design simple et moderne
  - Compteur centré
  - Boutons stylés 

# Exécution
  - Ouvrir le projet avec Android Studio
  - Lancer un émulateur ou connecter un téléphone
  - Cliquer sur Run ▶️
